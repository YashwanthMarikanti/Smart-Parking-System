# 🚗 Smart Parking System using Arduino

## 📌 Project Overview
This project is a Smart Parking System using Arduino to automate parking management using IR sensors and a servo motor.

## ⚙️ Features
- Automatic vehicle detection
- Real-time parking count
- Servo motor gate automation
- Parking full indication
- Low-cost implementation

## 🛠️ Components Used
- Arduino Uno
- IR Sensors
- Servo Motor
- LEDs
- Jumper Wires
- Power Supply
